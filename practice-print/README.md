# React 컴포넌트 Print

- npm i react-to-print
- npm i jspdf
- npm i html2canvas
- npm i @react-pdf/renderer

### 구현기능 :
1. Library 사용하지 않고 print 기능 구현

2. react-to-print를 이용하면 원하는 컴포넌트만 간단하게 print 가능

3. jspdf를 사용하면 pdf 파일을 download 가능

### 개선사항 :
